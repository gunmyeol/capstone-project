# 개인정보 보호정책 (Privacy Policy)

**ML-based Network Intrusion Detection System (IDS) - 캡스톤 디자인 프로젝트**

마지막 업데이트: 2026년 2월 22일

## 1. 개요

본 개인정보 보호정책은 "머신러닝 기반 네트워크 침입 탐지 시스템(ML-based Network Intrusion Detection System, IDS)" 캡스톤 디자인 프로젝트(이하 "프로젝트")에서 사용자의 개인정보를 어떻게 수집, 사용, 보호하는지 설명합니다.

## 2. 수집하는 개인정보

### 2.1 필수 정보
- **사용자 계정 정보**: 이름, 이메일 주소, 로그인 방식
- **인증 정보**: OAuth 토큰, 세션 정보
- **사용 로그**: 로그인 시간, 접속 기록, 활동 로그

### 2.2 서비스 제공을 위한 정보
- **업로드된 데이터셋**: 네트워크 트래픽 데이터 (개인식별 정보 제외)
- **모델 학습 기록**: 실험 파라미터, 학습 결과, 성능 지표
- **분석 결과**: 탐지된 위협 정보, 알림 기록

### 2.3 기술적 정보
- **IP 주소**: 접속 기록 추적 및 보안 목적
- **브라우저 정보**: User-Agent, 쿠키
- **시스템 로그**: 에러 로그, 접근 로그

## 3. 개인정보 사용 목적

개인정보는 다음의 목적으로만 사용됩니다:

1. **서비스 제공**: 사용자 계정 관리, 데이터 분석, 모델 학습
2. **보안**: 부정 접근 방지, 보안 위협 탐지
3. **개선**: 서비스 성능 개선, 사용자 경험 향상
4. **학술 목적**: 캡스톤 디자인 프로젝트 평가 및 논문 작성
5. **법적 의무**: 법령에 따른 의무 이행

## 4. 개인정보 보호 조치

### 4.1 기술적 보안
- **암호화**: 모든 통신은 HTTPS를 통해 암호화됨
- **데이터베이스 보안**: 암호화된 연결, 접근 제어
- **세션 관리**: 안전한 쿠키 설정 (HttpOnly, Secure, SameSite)

### 4.2 관리적 보안
- **접근 제어**: 필요한 사람만 데이터에 접근 가능
- **감시**: 비정상 접근 모니터링
- **교육**: 개인정보 보호 교육 실시

### 4.3 물리적 보안
- **서버 보안**: 클라우드 호스팅 제공자의 물리적 보안 조치 준수

## 5. 개인정보 보유 및 이용 기간

- **활성 사용자**: 서비스 이용 기간 동안 보유
- **휴면 계정**: 1년 이상 미접속 시 삭제 검토
- **법적 보관**: 법령에 따른 필수 보관 기간 준수

## 6. 개인정보 제3자 제공

원칙적으로 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외입니다:

- **법적 요구**: 법원 명령, 수사 기관 요청
- **서비스 제공자**: 클라우드 호스팅, 데이터베이스 서비스 제공자
- **학술 목적**: 익명화된 데이터의 학술 공개 (개인 식별 불가능)

## 7. 사용자의 권리

사용자는 다음의 권리를 가집니다:

- **접근권**: 자신의 개인정보 조회 요청
- **정정권**: 부정확한 정보 수정 요청
- **삭제권**: 개인정보 삭제 요청
- **이의제기권**: 개인정보 처리에 대한 이의 제기

이러한 요청은 프로젝트 관리자에게 이메일로 연락하여 진행할 수 있습니다.

## 8. 쿠키 및 추적 기술

본 프로젝트는 다음의 쿠키를 사용합니다:

- **세션 쿠키**: 사용자 인증 및 세션 유지
- **분석 쿠키**: 서비스 사용 패턴 분석 (선택적)

사용자는 브라우저 설정을 통해 쿠키를 거부할 수 있으나, 서비스 이용에 제약이 있을 수 있습니다.
